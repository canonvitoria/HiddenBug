Front-End

HTML

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar páginas web..

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a aparência e o layout de elementos HTML em páginas web.

JavaScript

JavaScript é uma linguagem de programação amplamente utilizada para adicionar interatividade e funcionalidade dinâmica às páginas web. Ela permite a manipulação de elementos HTML, o processamento de dados, a validação de formulários e a criação de efeitos visuais.

Back-end

PHP

PHP é uma linguagem de programação voltada para o desenvolvimento de aplicações para a web e para criar sites, favorecendo a conexão entre os servidores e a interface do usuário, e o aprendizado simples facilitam muito o caminho de profissionais.

Python

Conhecida por ser fácil de aprender e de utilizar, esta é uma linguagem simples e com boa legibilidade. Além disso, ela possui diversas aplicações, que vão desde páginas da web, passando por jogos até a dispositivos de Inteligência Artificial (IA).

Java

Java é uma linguagem de pro­gramação amplamente usada para codi­ficar aplicações Web. Ela tem sido uma esco­lha popular entre os desenvol­vedores há mais de duas décadas, com milhões de aplicações Java em uso hoje.

MySQL

O MySQL é um sistema de gerenciamento de banco de dados relacional (RDBMS) amplamente utilizado em aplicativos web e empresariais. O MySQL é um software de código aberto e oferece recursos robustos para lidar com diversas tarefas relacionadas a bancos de dados.

Framework

Bootstrap

O Bootstrap é uma ferramenta gratuita para desenvolvimento HTML, CSS e JS. Crie protótipos rápidamente ou aplicações completas com vários esquemas personalizáveis.

React

React, também conhecido como React JS, é uma popular biblioteca JavaScript de código aberto para criar aplicativos da web rápidos e escaláveis . É essencialmente uma estrutura JavaScript front-end que permite criar interfaces de usuário interativas de maneira rápida e fácil.

Angular

Angular é uma plataforma de aplicações web open source e front-end baseado em TypeScript liderado pela Equipe Angular do Google e por uma comunidade de indivíduos e corporações. Angular é uma reescrita completa do AngularJS, feito pela mesma equipe que o construiu.

GitHub

GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos privados e/ou Open Source de qualquer lugar do mundo.

Git

Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.

COMEÇO

Os estudantes André, Catiuscia, Jonatas e Vitória criaram a HiddenBug, uma plataforma de ensino para programadores iniciantes, como parte de seu TCC. Eles queriam facilitar o aprendizado de programação, oferecendo lições interativas e um ambiente de comunidade vibrante. Com uma equipe de desenvolvedores talentosos, eles construiram a plataforma, com aulas passo a passo, exercícios práticos e fóruns de discussão. O lançamento foi um sucesso, atraindo uma comunidade crescente de alunos. A HiddenBug se tornou uma referência no ensino de programação, capacitando e motivando novos programadores a dar seus primeiros passos com confiança.

VISÃO visão

Visão

A visão da HiddenBug é ser reconhecido como o principal destino para o aprendizado de programação, impulsionando a próxima geração de talentos em tecnologia por meio de uma abordagem inovadora, acessível e engajadora.

MISSÃO missão

Missão

A missão da HiddenBug é capacitar e inspirar aspirantes a programadores, oferecendo uma plataforma de ensino de alta qualidade, que promova o aprendizado prático, interativo e colaborativo, preparando-os para enfrentar os desafios do mundo da programação.

Valores valores

Valores

Os valores da HiddenBug são: acessibilidade, interatividade e comunidade, proporcionando um ambiente acolhedor e eficaz para o aprendizado de programação.
